This programming toolbox has three Python programs that utilize pyPDF in some way shape or form:

1. PyPDFTesting.py is a program that just tests a few functionality tools included in pyPDF, like rotating, extracting metadata, merging PDFs, and appending PDFs into one.
2. extractText.py takes a user input file and page number to extract the text from. Also has an option to extract all text from the PDF into a TXT file.
3. numberPages.py takes a user input file and numbers the pages with thier page number, saving it as a new PDF file.
